# TriviaMaster Bot  
A Discord bot for trivia games and quotes.  

## Features  
- `!trivia`: Answer a random trivia question (WIP: Only 2 questions).  
- `!quote`: Get a motivational quote (WIP: Limited list).  
- Basic conversational AI (via ChatterBot).
